# 빠른 시작 가이드

프로젝트를 실행하는 방법을 단계별로 설명합니다.

## 실행 방법

### 1단계: 의존성 설치 (최초 1회만)

터미널에서 프로젝트 폴더로 이동한 후 다음 명령어 실행:

```bash
npm install
```

이 명령어는 `package.json`에 정의된 모든 패키지를 설치합니다.

### 2단계: 개발 서버 실행

의존성 설치가 완료되면 다음 명령어로 개발 서버를 실행:

```bash
npm run dev
```

성공하면 다음과 같은 메시지가 표시됩니다:
```
  ▲ Next.js 14.x.x
  - Local:        http://localhost:3000
  - Ready in X seconds
```

### 3단계: 브라우저에서 확인

브라우저를 열고 다음 주소로 접속:
- **메인 테스트 페이지**: http://localhost:3000
- **관리자 대시보드**: http://localhost:3000/admin

## 주의사항

### Supabase 없이 실행하는 경우

Supabase를 설정하지 않은 상태에서도 앱은 실행됩니다. 하지만:
- ⚠️ 테스트 완료 후 데이터 저장이 작동하지 않습니다
- ⚠️ 관리자 페이지에서 데이터를 볼 수 없습니다
- ✅ 테스트 진행 자체는 정상적으로 작동합니다

### Supabase 설정이 필요한 경우

데이터 저장 기능을 사용하려면:
1. `.env.local` 파일 생성
2. Supabase 프로젝트 설정 (SUPABASE_SETUP.md 참고)
3. 환경 변수 입력

## 자주 사용하는 명령어

| 명령어 | 설명 |
|--------|------|
| `npm install` | 의존성 설치 (최초 1회) |
| `npm run dev` | 개발 서버 실행 |
| `npm run build` | 프로덕션 빌드 |
| `npm run start` | 프로덕션 서버 실행 |
| `npm run lint` | 코드 린팅 |

## 개발 서버 중지

터미널에서 `Ctrl + C` (Mac/Linux) 또는 `Ctrl + C` (Windows)를 눌러 개발 서버를 중지할 수 있습니다.
